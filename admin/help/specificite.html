Spécificités

Ce système de gestion de contenu (CMS, Content Management System) est fondé sur la séparation entre les données (le texte, les articles...) et le design.

Voici ses caractéristiques principales:

  • Séparation du code PHP, des données (les articles) et du design (les gabarits - templates en anglais - et les fichiers de style css.
  • Réécriture d'url (les liens sont plus lisibles et permettent un meilleur référencement.
  • Flux de syndication (atom).
  • Cache navigateur (limité avec free)
  • Cache html
  • Gestion des droits d'accès
