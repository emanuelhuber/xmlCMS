Créer un nouveau diaporama

Pour créer un nouveau diaporama, chargez vous-même les fichiers via FTP sur le site. Déposez dans le dossier "votre-site/data/plugin/diaporama/" un dossier ayant le nom de votre diaporama (en minuscule, que des lettres et des tirets, pas d'espaces) contenant:

  • Un fichier texte nommé "config.txt" contenant la configuratio Javascritp (voir sur ce site pour les détails).
  • Un dossier nommé "images" contenant uniquement les images du diaporama. Ces images devraient être toutes de la même taille et cette taille doit correspondre à l'agencement de votre site.
