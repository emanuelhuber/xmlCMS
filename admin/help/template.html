Convention de nommage des variables : les variables de texte doivent commencer par L_, les urls par U_ , les urls javascript par UA_, les textes javascript par LA_, les données système par S_ et le reste à votre convenance (si vous ne respectez pas ces conventions, sauf pour les accolades, ça n'entrainera pas d'erreur, c'est juste pour améliorer la lisibilité de votre code) http://forums.phpbb-fr.com/documentation-phpbb3/sujet154167.html#p1178206
